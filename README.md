# Job Application Tracker

A full-stack CRUD app, tracking job applications.

## Features:
- CRUD - able to add, read, edit and delete application inputs
- Colour coded status
- Error handling
- Responsive layout using Tailwind CSS
- TypeScript type checking

## Tech
- Next.js
- React.js
- Supabase Database
- TypeScript
- Data validation (Zod)
- Tailwind CSS
- Deployed on Vercel

## Run the app locally:
- Clone the repo from GitHub
- Run `npm install`
- Add your Supabase key `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- Run `npm run dev`
